About Me

Saya adalah seorang web developer pemula berusia 19 tahun dengan semangat tinggi dalam dunia teknologi dan pemrograman. Latar belakang pendidikan saya di bidang Sistem Informasi memberi saya dasar yang kuat dalam memahami konsep-konsep penting yang berkaitan dengan pengembangan web.

Minat saya dalam coding dimulai sejak SMA, di mana saya belajar HTML, CSS, dan JavaScript untuk pertama kalinya. Sejak saat itu, saya terus mengasah keterampilan saya melalui kursus online, workshop, dan proyek-proyek pribadi yang saya kerjakan. Saya percaya bahwa dengan dedikasi dan ketekunan, saya dapat terus berkembang sebagai seorang profesional di dunia teknologi informasi.
